# 🚀 Тестовий Git Репозиторій

Ласкаво просимо до навчального проекту для опанування основ роботи з Git та веб-розробки!

## 📋 Опис проекту

Цей репозиторій створений для демонстрації навичок роботи з контролем версій та створення базових HTML-сторінок. Тут ви знайдете приклади структури веб-сторінок та стилізації.

## 🏗️ Структура проекту

- `index.html` — Головна сторінка з привітанням "Hello world".
- `index2.html` — Додаткова сторінка з текстовим наповненням (Lorem Ipsum).
- `style.css` — Файл зі стилями для оформлення сторінок.
- `README.md` — Цей файл з документацією.

## 🛠️ Технології

- **HTML5** — Структура контенту.
- **CSS3** — Стилізація.
- **Git** — Система контролю версій.

## 🚀 Як запустити

Просто відкрийте файл `index.html` у вашому улюбленому браузері!
